# Hexagonal Linkage Volume Calculator
This is a C# ASP.NET MVC web application designed to calculate the volume of an irregular solid (Hexagonal Linkage) using the Fluid Displacement Method.

## Features:
- Archimedes' Principle Logic
- AI-Assisted UI Design
- Pure C# Logic (MVC Pattern)

### 🛠 How it Works
1. Enter the **Initial Water Level (V1)**.
2. Enter the **Final Water Level (V2)**.
3. The system calculates the volume using the formula: **Volume = V2 - V1**.
